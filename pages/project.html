🐍 Python Projects

  • Railway Reservation System Python SQLite CSV
    A comprehensive CLI-based Railway Reservation System inspired by IRCTC. Features include multi-passenger booking with individual passenger details, partial seat confirmation (confirmed seats filled first, then waitlist), Tatkal booking with 25% quota reservation and 40-50% surcharge, global waitlist numbering per train/date/class, automatic waitlist promotion on cancellation, PNR status check, cancellation with refund calculation, admin panel with revenue reports (by date, train, class), ticket counter mode for booking on behalf of passengers, and a dataset of 10,000 trains generated programmatically.
    GitHub
  • Bank Management System Python SQLite SHA256 smtplib
    A feature-rich banking application with customer account creation (Savings/Current), deposits, withdrawals with minimum balance enforcement, money transfers between accounts, mini statement showing last 10 transactions, loan management with EMI calculation using standard financial formulae (Personal, Home, Education, Vehicle loans), account freeze/unfreeze by admin, forgot PIN via email OTP, change PIN, close account with validation, admin controls with account search, CSV report generation for accounts, loans and transactions. Implements SHA256 password hashing for security.
    GitHub
  • Student Result Management System Python SQLite CSV
    A student result management application with complete CRUD operations — add, view, update and delete student records. Features subject-wise analysis showing average, highest and lowest marks per subject along with top performing student, automatic grade and percentage calculation (First Division, Second Division, Fail), search by roll number, multi-admin support with add/remove admin functionality, password hashing using SHA256, CSV export of results, and a fully menu-driven interface. Packaged as a standalone Windows executable using PyInstaller.
    GitHub
